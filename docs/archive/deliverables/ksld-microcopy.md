KSLD — Microcopy Pack (Buttons, Forms, Errors, 404)

Buttons & CTAs
- Primary: "Apply Now"
- Secondary: "Explore Programs"
- Secondary alt: "Download Brochure"
- Tertiary: "Book Campus Visit"
- Small card CTA: "Learn More"
- Inline link: "See Success Stories"

Form labels & placeholders
- Full name
  - Placeholder: "First Last"
- Email
  - Placeholder: "name@example.com"
- Phone
  - Placeholder: "+91 9XXXXXXXXX"
- Message
  - Placeholder: "How can we help? Tell us briefly."

Helper text
- Email helper: "We’ll only use this to contact you about admissions."
- Phone helper: "For quick follow-ups on your application."

Validation messages (friendly, actionable)
- Required field: "This field is required."
- Invalid email: "Please enter a valid email address (name@example.com)."
- Invalid phone: "Please enter a valid Indian phone number, e.g. +91 9XXXXXXXXX."
- Too short message: "Tell us a bit more so we can help." 

Form success
- Submission success: "Thanks — we’ve received your message. Admissions will respond within 3 business days."

404 page
- Headline: "Page not found"
- Body: "We couldn’t find that page. Try searching or return to the homepage." 
- CTAs: "Go Home" / "Contact Admissions"

Search empty state
- "No results — try a different keyword or browse our programs."

Footer microcopy
- Short tagline: "Where Leaders and Founders Are Built."
- Contact line: "admissions.ksld@kiit.ac.in | +91-674-2725113"

Accessibility notes
- All CTAs include aria-labels where text could be ambiguous (e.g., "Apply Now — opens application form").
- Form error text should be linked to the input via `aria-describedby` and focus should move to the first invalid input on submit.

Localization
- Dates & currency: use local format (DD MMM YYYY) and INR (₹) with large numbers grouped (1,300 Cr = ₹1,300 Cr+).

Small copy examples (for components)
- Card meta: "Class of 2023 • Founder & CEO"
- Stat label: "Startups Supported"
- Mentor callout: "300+ mentors from industry and policy"

Delivery
- Paste these microcopy items into your design system or component library as default text values for buttons, inputs, and error states.
