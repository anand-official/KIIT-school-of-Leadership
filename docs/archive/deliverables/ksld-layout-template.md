Design Token References (for all artboards)

Color system:
- Background: #FFFFFF (white) or #F5F5F5 (light gray)
- Text primary: #000000
- Text secondary: #666666
- Accent: #2D5F3F (KIIT Green)
- Border: #E8E8E8
- Dark section: #1A1A1A or #2C2C2C

Typography:
- H1: Playfair Display, 52px, 700, 64px line-height
- H2: Playfair Display, 40px, 600, 48px line-height
- H3: Inter, 20px, 600, 28px line-height
- Body: Inter, 16–18px, 400, 24–28px line-height
- Caption: Inter, 13–14px, 400, 18px line-height

Spacing scale:
- 8px, 12px, 16px, 24px, 32px, 40px, 48px, 60px, 80px, 100px, 120px

Border & shadows:
- Rounded corners: 6–8px (cards), 4px (small elements)
- Shadow (cards): 0px 4px 20px rgba(0,0,0,0.08)
- Shadow (hover): 0px 12px 32px rgba(0,0,0,0.12)

Responsive breakpoints:
- Desktop: 1440px
- Tablet: 1024px
- Mobile: 375px

Motion (reference only, implement in code):
- Transition duration: 300ms
- Easing: ease-in-out (cubic-bezier)
- Hover effects: Subtle lift (2–4px) + shadow increase
